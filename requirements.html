System Requirements

General Requirements

  • Webserver with PHP enabled.
  • PHP configured with your SQL drivers (Postgres, MySQL, SQL Server, Oracle, SQLite, etc.).

Installation Requirements — Windows Server

Windows IIS Web Server with PHP Manager and PHP 8.x
  • 1) Go to PHP.net and download the latest ZIP. Create a directory under C:\Program Files\PHP\V8.x and extract the ZIP.
  • 2) In Windows Server Manager, add .NET 3.5 and 4.5, and include CGI Tools.
  • 3) Download PHP Manager for Windows (MIT Open Source), version 2.x or later.
  • 4) In PHP Manager, register the new version of PHP located at C:\Program Files\PHP\V8.x\php.exe or php.dll.

Installation Requirements — Linux

Apache or Nginx with PHP / PHP‑FPM
  • Depending on your Linux distribution, install PHP using apt-get install php (Debian/Ubuntu) or yum/dnf install php (RedHat/CentOS/Oracle Linux).
  • We recommend installing Webmin, which supports Apache out of the box and Nginx with custom configuration options.
  • Nginx requires a custom PHP‑FPM configuration. See DigitalOcean’s tutorial for guidance.
  • For beginners, cPanel servers are recommended because they include complex PHP configurations already working with Apache.

Installation Requirements — macOS

Apache with PHP
  • 1) Older versions of macOS include PHP with Apache. Uncomment PHP in /etc/apache2/httpd.conf.
  • 2) Newer macOS versions do not include PHP. Install MacPorts or Homebrew, then run port install php or brew install php.
  • 3) Enable PHP in Apache using a2enmod mod_php (varies by version). Many tutorials are available online.
